Slack status update scheduler task.

This class provides a scheduler task to update the status of a Slack user. It can be used for example to restore a specific status.

bool run(string|null $userId = null, string|null $statusText = null, string|null $statusEmoji = null)

Set a status text and status emoji of a given Slack user.

Sets the status text and status emoji of a given Slack user in case the user is already authenticated. If not, this task cannot be used to update the status and the user needs to authenticate himself first. If both status text and emoji are empty, the task won't change the status. This is also the case if a status is currently set for the specified user.

Parameters

string|null $userId User id of the Slack user whose status should be updated
string|null $statusText New status message for the given Slack user
string|null $statusEmoji New status emoji for the given Slack user

Return Value

bool true if the execution was successful, false otherwise

Exceptions

AuthenticationException if the user is not authenticated or needs to re-authenticate himself
InvalidRequestException if API request failed or contains an invalid answer
MissingParameterException if the $userId parameter is not set or empty
