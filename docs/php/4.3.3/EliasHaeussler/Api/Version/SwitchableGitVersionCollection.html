SwitchableGitVersionCollection

class SwitchableGitVersionCollection extends GitVersionCollection (View source)

Switchable Git version collection.

Git version collection class which allows to switch between versions aka Git tags. This class is necessary as the default {see GitVersionCollection} class does not allow to safely switch between Git tags while the working directory is not clean. It solves this issue by stashing the build files which makes it possible to unstash them after the build process has finished. This way, all build files will be kept during the whole build process.

protected switchVersion(Version $version)

Switch to given version.

Switches to the given version by checking out the appropriate Git tag or branch. This method also provides functionality to stash the previous build files in order to make the working directory clean. Doing this allows the parent method to safely switch between versions.
