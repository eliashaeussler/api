SchedulerService

class SchedulerService (View source)

Task scheduler service.

This class provides a service to schedule and execute tasks. Tasks are working processes which need to be executed at a given time in the future. If the execution time was met, the scheduler service can be used to execute the tasks. Each task needs to extend the {see AbstractTask} class in order to serve as scheduler task.

static bool scheduleTask(string $className, DateTime $execution, array $arguments = [])

Schedule a specific task.

Stores a specific task in the database in order to make them execute at the given execution time. It's important that the task meets the requirements, such as extending the {see AbstractTask} class and serving the correct arguments needed for executing the appropriate task method. Also, the execution time has to be in the future.

Parameters

string $className Name of the class to be used to execute the task
DateTime $execution Scheduled task execution time
array $arguments Arguments which will be passed to the task method

Return Value

bool true if the task was successfully scheduled, false otherwise

Exceptions

ClassNotFoundException if the {see ConnectionService} class is not available
InvalidClassException if the task class does not extend the {see AbstractTask} class
InvalidExecutionTimeException if the scheduled execution time is not in the future
InvalidMethodSignatureException if the number of arguments does not match the number of task method arguments
ReflectionException if the task class does not exist
Exception if the {see DateTime} object cannot be instantiated

static array getScheduledTasks(string|null $className = null, int|null $uid = null, int $limit = 20, bool $ignoreExecutionTime = false)

Get currently scheduled tasks.

Returns tasks which are currently scheduled for execution. Depending on the provided arguments, this method either returns all tasks or only tasks matching the given uid and/or the given class name. The class name can either be a FQN or the full class name relative to the EliasHaeussler\Api\Task namespace.

Parameters

int|null $uid Uid of a specific task to be returned
int $limit Maximum numbers of tasks to be executed at this iteration
bool $ignoreExecutionTime Define whether to ignore exeuction time and return all tasks matching the constraints

Return Value

array Result set of the tasks which are scheduled for execution
